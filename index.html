What is Css?

  1. Css stands for Cascading Style Sheets.

  2. Css describe how HTML elements are to be displayed on the screen, paper, or in the other media.

  3. Css is not necessary Css saves lots of work. It can control the layout of multiple web pages all at once

  4. External stylesheets are stored in CSS files.

CSS can be addded to HTML documents in 3 ways:

  • Inline - by using the style attribute inside the HTML elementsl
  • Internal - by using a style element in the head section
  • External - by using the link element to link to an external CSS file
  • The most common way to add CSS, it is to keep the styles in external CSS files. However, in this tutorial we will use inline and internal style,because this is easier to demonstarte, and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is also special in that you can use special URL to link directly to an element and it's used by javScript.

In CSS, selectors are used to target a specific element or range of elements on the web page. Once an element has been targeted, a style or set of style can be applied to the element.

There is wide range of selectors are available. Two of the most commonly used are class and ID. Both are used to target elements to which a style should be applied.
